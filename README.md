# How to develop the project
## Web Services
1. Go to **ws** folder
2. Run *docker-compose up --build --detach*
3. Run *docker attach ws_client_1*
4. Input 1, 2, 3, X to interact

## Micro Services
1. Go to **ms** folder
2. Run *docker-compose up --build --detach*
3. Run *docker attach ms_client_1*
4. Input 1, 2, 3, X to interact
